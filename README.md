# Crossy-Pandas
- Developed a fully playable, 2D arcade game from scratch using Java Swing, managing end-to-end development from custom pixel art rendering to core game logic.
- Designed a scalable object-oriented architecture leveraging abstract inheritance and enums to efficiently handle polymorphic rendering, sprite directional rotation, and strict collision detection.
- Utilized thread-safe data structures for player and entity coordinate tracking to prevent race conditions during rapid keyboard inputs.﻿
- Implemented procedural level generation algorithms to construct unique, randomized game boards.
- Engineered a dynamic game loop utilizing javax.swing.Timer to handle real-time UI repainting, continuous obstacle animation, and screen scrolling at a consistent frame rate.
